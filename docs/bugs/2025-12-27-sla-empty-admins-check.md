# Проверка на пустой список админов в SLA

**Дата:** 2025-12-27
**Источник:** TD-011

## Проблема

При SLA reminder, если `getGroupAdmins()` возвращает пустой массив (все админы — боты), сообщение отправлялось без @mentions и никто не получал уведомление. При этом не было никакого лога об этой ситуации.

## Решение

Добавлена проверка на пустой список админов с error-уровнем логом:

```typescript
if (admins.length === 0) {
  logger.error({ userId, topicId, level },
    'No human admins found for SLA reminder - notification will be ineffective');
}
```

## Изменённые файлы

- `src/jobs/sla.worker.ts` — добавлена проверка пустого списка

## Важно для разработчика

- Этот лог указывает на проблему конфигурации группы
- SLA reminder всё равно отправится, но без упоминаний
