# Widget: Текст кнопки подключения Telegram

**Дата:** 2025-12-31
**Источник:** TD-038

## Проблема

Текст кнопки "Подключить" в виджете чата не точно отражал действие. Отсутствовало пояснение преимуществ использования Telegram-бота напрямую.

## Решение

1. Изменён текст кнопки с "Подключить" на "Перейти"
2. Добавлен пояснительный текст после кнопки

## Изменённые файлы

- `chat-widget/src/ui/telegram.ts` — изменён текст кнопки, добавлен span с описанием
- `chat-widget/src/styles/base.css` — добавлены стили для `.chat-telegram__description`

## Важно для разработчика

Структура компонента TelegramLink:
```
.chat-telegram (flex-wrap: wrap)
├── .chat-telegram__text — "Продолжить в Telegram?"
├── .chat-telegram__btn — кнопка "Перейти"
└── .chat-telegram__description — пояснительный текст (flex-basis: 100%)
```

Пояснительный текст занимает всю ширину контейнера благодаря `flex-basis: 100%` и переносится на новую строку.
