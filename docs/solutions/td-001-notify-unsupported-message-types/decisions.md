# Решения по TD-001

## Решение: Ранний возврат при неподдерживаемом типе

**Контекст:** Нужно уведомить пользователя и прекратить обработку.

**Решение:** После проверки `mirrorResult === null` отправляем сообщение и делаем `return`, чтобы не выполнять дальнейшую логику (отмена таймеров, смена статуса).

**Обоснование:** Если сообщение не было доставлено в поддержку, нет смысла менять статус тикета или отменять таймеры — это могло бы привести к некорректному состоянию.
