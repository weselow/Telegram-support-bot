# Задача 005: Создание топика при первом сообщении

**Этап:** 2 - Основной функционал
**Статус:** DONE
**Приоритет:** Высокий

## Описание

При первом сообщении пользователя создавать топик в супергруппе и сохранять связь в БД.

## Подзадачи

- [x] Сервис для работы с топиками (src/services/topic.service.ts):
  - [x] Создание топика через Telegram API (createForumTopic)
  - [x] Формирование названия топика (имя пользователя или ID)
- [x] Сервис для работы с тикетами (src/services/ticket.service.ts):
  - [x] Создание пользователя в БД
  - [x] Запись события "opened" в ticket_events
- [x] Репозитории (src/db/repositories/):
  - [x] user.repository.ts
  - [x] event.repository.ts
- [x] Обработчик первого сообщения:
  - [x] Проверка существования пользователя
  - [x] Создание топика если нет
  - [x] Отправка карточки тикета в топик
  - [x] Закрепление карточки
