# 025: Сообщение в нерабочее время

**Статус:** TODO
**Приоритет:** Низкий
**Источник:** TD-038 (отложенный текст копирайтера)

## Описание

Отправлять пользователю сообщение, если он пишет в нерабочее время поддержки.

## Текст от копирайтера

> "Сейчас мы не на связи. Оставьте сообщение — ответим в рабочее время (Пн–Пт, 9:00–18:00 МСК)."

## Требования

- [ ] Определить расписание работы поддержки (конфиг)
- [ ] Учитывать часовой пояс (МСК)
- [ ] Отправлять сообщение при первом обращении в нерабочее время
- [ ] Добавить текст в `config/messages.json`
- [ ] Опционально: добавить расписание в `config/config.json`

## Технические заметки

- Расписание: `{ workDays: [1,2,3,4,5], startHour: 9, endHour: 18, timezone: 'Europe/Moscow' }`
- Проверка при создании тикета или получении сообщения

## Acceptance Criteria

- [ ] Пользователь получает сообщение в нерабочее время
- [ ] Расписание настраивается через конфиг
- [ ] Корректно работает с часовым поясом МСК
