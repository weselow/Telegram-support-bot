# 024: Сообщение при задержке ответа

**Статус:** TODO
**Приоритет:** Низкий
**Источник:** TD-038 (отложенный текст копирайтера)

## Описание

Отправлять пользователю сообщение, если поддержка не ответила в течение N минут.

## Текст от копирайтера

> "Извините за ожидание. Ваш вопрос в очереди — ответим, как только освободимся."

## Требования

- [ ] Определить триггер: через сколько минут отправлять (например, 15-20 мин)
- [ ] Интегрировать с существующей SLA-системой или создать отдельный таймер
- [ ] Отправлять только один раз за тикет
- [ ] Не отправлять если тикет уже закрыт или получил ответ
- [ ] Добавить текст в `config/messages.json`

## Технические заметки

- Возможно использовать BullMQ delayed jobs (уже есть в проекте)
- Проверить пересечение с SLA-таймерами (013-sla-timers)

## Acceptance Criteria

- [ ] Пользователь получает сообщение при задержке
- [ ] Сообщение отправляется только один раз
- [ ] Таймер отменяется при ответе поддержки
