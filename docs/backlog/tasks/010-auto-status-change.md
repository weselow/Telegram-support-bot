# Задача 010: Автоматическая смена статусов

**Этап:** 3 - Расширенный функционал
**Статус:** TODO
**Приоритет:** Средний

## Описание

Автоматически менять статус тикета при определённых событиях.

## Подзадачи

- [ ] Автосмена при ответе сотрудника:
  - [ ] Если статус "Новый" → "В работе"
  - [ ] Обновление карточки
  - [ ] Запись события
- [ ] Автосмена при ответе клиента:
  - [ ] Если статус "Ждём клиента" → "В работе"
  - [ ] Обновление карточки
  - [ ] Запись события
- [ ] Кнопка "Спасибо, вопрос решён" для клиента:
  - [ ] Отображение в DM
  - [ ] Обработчик callback
  - [ ] Статус → "Закрыт"

## Зависимости

- [008-ticket-card-buttons](./008-ticket-card-buttons.md)

## Результат

- Статусы меняются автоматически по правилам
- Клиент может закрыть тикет сам
- Все изменения логируются
