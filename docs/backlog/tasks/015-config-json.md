# Задача 015: Конфигурация из config.json

**Этап:** 5 - Полировка
**Статус:** TODO

## Описание

Вынести тексты сообщений и настройки в config.json для изменения без перекомпиляции.

- [ ] Создание config/messages.json:
  - [ ] Приветственное сообщение
  - [ ] Подтверждение создания тикета
  - [ ] Запрос телефона
  - [ ] Подтверждение телефона при переоткрытии
  - [ ] Уведомления о смене статуса
  - [ ] SLA-напоминания
  - [ ] Сообщение о блокировке бота
  - [ ] Кнопки (тексты)
- [ ] Загрузчик конфига (src/config/messages.ts):
  - [ ] Чтение JSON
  - [ ] Валидация структуры
  - [ ] Hot-reload (опционально)
- [ ] Настройки SLA-таймеров в конфиге:
  - [ ] sla_first_reminder_minutes: 10
  - [ ] sla_second_reminder_minutes: 30
  - [ ] sla_breach_minutes: 120
  - [ ] autoclose_days: 7
