# TD-032: Chat Widget — Переключатель режимов modal/drawer

**Приоритет:** Low
**Статус:** Идея
**Модуль:** chat-widget

## Описание

Добавить в заголовок чата маленький переключатель режимов отображения (modal/drawer), чтобы пользователь мог выбрать удобный для себя вариант.

## Текущее поведение

- `variant: 'auto'` выбирает режим автоматически (desktop → modal, mobile → drawer)
- Пользователь не может изменить режим

## Предлагаемое решение

1. В правой части заголовка добавить иконку-переключатель
2. При клике переключать между modal ↔ drawer
3. Сохранять выбор в localStorage/cookie
4. При следующем визите использовать сохранённый выбор (если `variant: 'auto'`)

## UI

```
┌─────────────────────────────────────┐
│ Поддержка DellShop    [⬜] [—] [✕] │
│                        ↑            │
│                   переключатель     │
└─────────────────────────────────────┘
```

Иконки:
- `⬜` — modal (окно)
- `▭` — drawer (панель)

## Связанные файлы

- `chat-widget/src/ui/header.ts` — добавить кнопку
- `chat-widget/src/utils/storage.ts` — сохранение выбора
- `chat-widget/src/widget.ts` — логика переключения
