# TD-049: Widget Drag&Drop и отмена загрузки

**Статус:** TODO
**Приоритет:** Низкий
**Источник:** Task 023 (Web Chat File Upload)

## Проблема

В текущей реализации загрузки файлов в веб-чате отсутствуют:
1. Drag&Drop для удобного перетаскивания файлов
2. Возможность отменить загрузку в процессе

## Требования

### Drag&Drop
- Зона перетаскивания поверх чата
- Визуальная индикация при перетаскивании файла
- Поддержка множественных файлов (загрузка по одному)

### Отмена загрузки
- Кнопка отмены во время загрузки
- `AbortController` для отмены XHR запроса
- Очистка preview при отмене

## Изменяемые файлы

- `chat-widget/src/ui/input.ts` - Drag&Drop listeners
- `chat-widget/src/transport/http.ts` - AbortController
- `chat-widget/src/styles/base.css` - стили drop zone
