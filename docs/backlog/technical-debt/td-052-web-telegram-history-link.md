# TD-052: Ссылка на историю сообщений при переходе Web → Telegram

**Дата:** 2026-01-03
**Статус:** BACKLOG
**Приоритет:** LOW

## Описание

При сценарии 3 (Web → Telegram) в приветственном сообщении нужно добавить ссылку на историю сообщений, чтобы пользователь мог просмотреть полную переписку.

## Контекст

Когда пользователь начинает чат на сайте, а затем переходит в Telegram:
1. Telegram-аккаунт привязывается к web-сессии
2. История сообщений сохраняется
3. Пользователь получает приветственное сообщение: "Ваш Telegram привязан к чату на сайте. История сообщений сохранена."

Было бы полезно добавить в это сообщение ссылку для просмотра истории.

## Задачи

- [ ] Разработать механизм генерации ссылки на историю для конкретного пользователя
- [ ] Добавить endpoint для отображения истории (возможно, отдельная web-страница)
- [ ] Обновить приветственное сообщение в messages.json с плейсхолдером для ссылки
- [ ] Обновить onboarding flow для вставки ссылки в сообщение
