# TD-038: Изменить текст кнопки подключения в виджете

**Статус:** TODO
**Приоритет:** Low
**Тип:** UX копирайтинг

## Описание

@screens\Screenshot_1.png - Изменить текст на кнопке подключения Telegram в виджете:
- Было: "Подключить"
- Стало: "Перейти"

После кнопки добавить пояснительный текст:
> Сможете отправлять картинки, файлы и голосовые сообщения, а также всегда вернуться к разговору.

## Причина

Текст "Перейти" точнее отражает действие (переход в Telegram), а дополнительный текст объясняет преимущества использования бота напрямую.

## Затронутые файлы

- `widget/` — компоненты виджета с кнопкой подключения

## Критерии готовности

- [ ] Текст кнопки изменён на "Перейти"
- [ ] Добавлен пояснительный текст после кнопки
- [ ] Текст корректно отображается на разных размерах экрана
