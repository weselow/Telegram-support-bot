# TD-047: Прокси для аватара бота

**Приоритет:** Низкий
**Источник:** Code Review TD-040 (Security)

## Проблема

URL аватара бота содержит токен бота:
```
https://api.telegram.org/file/bot{TOKEN}/{file_path}
```

Этот URL передаётся клиенту, что теоретически раскрывает токен.

## Риск

**Низкий:**
- file_path временный (~1 час)
- Токен один на сервис, не даёт доступа к другим ботам
- URL используется только для загрузки изображения

## Решение (опционально)

1. Создать эндпоинт `/api/chat/bot-avatar`
2. Проксировать запрос на Telegram File API
3. Кэшировать изображение локально

## Критерии готовности

- [ ] Эндпоинт создан
- [ ] Изображение кэшируется
- [ ] Widget использует прокси вместо прямого URL
